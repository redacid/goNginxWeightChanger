# goNginxWeightChanger

## help
    goNginxWeightChanger -h

## Show parsed configuration file
    goNginxWeightChanger -command showConfig

## Change weight

### With write changes

```
goNginxWeightChanger -command changeWeight -writeWeightChanges yes
``` 

### Preview only

```
    goNginxWeightChanger -command changeWeight -writeWeightChanges yes
```
